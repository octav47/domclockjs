Теперь у Radio Hustle есть API, текущая версия: 0.2.1 Подключить API: 
                        
                            <script src="http://octav47.github.io/projects/radioapi.min.js"></script>
Чтобы получить информацию о танцоре по его номеру: 
                        
                            RadioAPI.getUser(id, async, success, error)
id -- номер танцора,
async -- true - асинхронный запрос, false - синхронный,
success -- callback при ответе 200 (для синхронных запросов не нужен),
error -- callback при любом другом ответе

В примерах все запросы асинхронные.
